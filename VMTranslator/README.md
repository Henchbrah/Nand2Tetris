This repository contains code for the second of five software development projects presented in The Elements of Computing Systems: Building a Modern Computer from First Principles by Noam Nisan and Shimon Schocken, a Virtual Machine translator for the Hack platform. This is the first step toward building a compiler for a typical object-based high-level programming language. For more information on The Elements of Computing Systems please visit http://www.nand2tetris.org/
